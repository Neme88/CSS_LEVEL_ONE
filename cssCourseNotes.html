CSS Level Part One

In this session we will cover the basics of CSS and how to link CSS file to our HTML file

CSS is a style sheet language used for describing the presentation of a document written in HTML or XML

Example below is how to link CSS file to an html file

<link rel="stylesheet" href="part1_master.css">

In this example the link tag and its rel attribute is used to define that we are linking a CSS file while the href is use input the path to the css file it can be relative or absolute path depending on the location of the cs file in the machine.

When your CSS file is properly link to html file all the styling changes you make in the css file reflect on the tags targeted

You can also style your html directly in the same file which is called inline styling however the best practice is to have a separate file for css code this for modularity

Comment in CSS

CSS General form
